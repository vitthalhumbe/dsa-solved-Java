# 61. Rotate List
Given the head of a linked list, rotate the list to the right by k places.

# my code 
```java
class Solution {
    public ListNode rotateRight(ListNode head, int k) {
        
        if (head == null) return null;
        int n = 0;
        ListNode curr = head;
        while (curr.next != null) {
            n++;
            curr = curr.next;
        }
        n++;
        k  = k % n;
        curr.next = head;
        ListNode prev = curr;
        int pos = n - k;
        for (int i = 0; i < pos; i++) {
            head = head.next;
            prev = prev.next;
        }
        prev.next = null;
        return head;

    }
}
```
